# Security Policy

## Supported Versions

tomix-cli is pre-1.0. Only the latest release receives security fixes.

## Reporting a Vulnerability

Please do not open a public issue for security problems.

Report vulnerabilities privately via GitHub:
[Report a vulnerability](https://github.com/bgarcevic/tomix-cli/security/advisories/new)

You can expect an initial response within a few days. Once a fix is available,
the vulnerability will be disclosed in the release notes.
